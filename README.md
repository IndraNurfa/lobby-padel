# Project Documentation

## Project Overview
This project is a comprehensive solution designed to enhance the experience of players and organizers in padel sports. It serves as a platform for managing reservations, player coordination, and event organization.

## Architecture
The architecture follows a microservices design, ensuring scalability and independent deployment of different functionalities. Key components include:
- **Frontend**: A responsive web application built with React.
- **Backend**: A RESTful API using Node.js and Express.
- **Database**: MongoDB for storing user data, reservations, and match details.

## Full-Stack Tech Stack
- **Frontend**: React, Redux, CSS-in-JS, Axios
- **Backend**: Node.js, Express, JWT for authentication
- **Database**: MongoDB
- **Others**: Docker for containerization, Jest for testing

## Features
- User authentication and profile management
- Padel court reservation system
- Event organization and scheduling
- Real-time notifications and updates
- User-friendly interface with mobile compatibility

## Installation Guide
1. **Clone the repository**: `git clone https://github.com/IndraNurfa/lobby-padel.git`
2. **Navigate to the project directory**: `cd lobby-padel`
3. **Install dependencies**: Run `npm install` for both the frontend and backend directories.
4. **Set up environment variables**: Create a `.env` file based on the provided template and fill in your credentials.
5. **Run the application**: Start both the frontend and backend servers by running `npm start` in their respective directories.
